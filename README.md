# abuadnan.dev portfolio

A premium, static, responsive single-page portfolio for Abu Adnan built with semantic HTML, CSS, and vanilla JavaScript.

## Structure

- index.html — main single-page layout
- assets/css/styles.css — visual design, responsive layout, and animation styles
- assets/js/main.js — content rendering, theme toggle, navigation, and reveal animations
- assets/images/ — branding and hero visuals
- favicon.svg — favicon asset
- robots.txt and sitemap.xml — SEO basics

## Customization

Update copy in the data object inside assets/js/main.js to change experience, skills, and projects without touching the layout.

## Deployment

This site is ready for static hosting such as Cloudflare Pages.
